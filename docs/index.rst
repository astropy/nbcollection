nbcollection
############

Execute and convert collections of Jupyter notebooks to static HTML

Tools for building collections of Jupyter notebooks into web pages for public consumption.

This project serves as a thin wrapper around nbconvert to enable converting and executing directories or directory
structures full of Jupyter notebooks to static HTML pages.

nbcollection-ci
+++++++++++++++

nbcollection-ci provides a set of tools to streamline the Scientific Review process of Jupyter Notebooks in astronomy.
Reporting for bugs and feature requests can be found on `Github`_


.. _Github: https://github.com/astropy/nbcollection/issues
